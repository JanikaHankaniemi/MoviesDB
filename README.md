# MoviesDB
## Needed tools:
VisualStudio (development done in VS 2022)
MongoDB instance

## How to start the API

Debug > Start Debugging  
Shortcut: `F5`

OR

Debug > Start Without Debugging  
Shortcut: `Ctrl`+`F5`

OR

In cmd, PowerShell or other terminal:

```ps
cd <your git folder>\MoviesAPI
dotnet run
```
